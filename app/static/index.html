Single Term — Best when you know exactly what to search. Use for quick tests or a specific keyword (e.g. "Zahnarzt" finds dentists). Fast and cheap, but may miss businesses that use a different name for the same service.
Category Bundle — Best for thorough coverage. A bundle contains 10-40 related terms that catch businesses using different names for the same thing. E.g. a "Dentists" bundle searches "Zahnarzt", "Zahnklinik", "Kieferorthopäde", etc. Finds 3-5x more businesses than a single term, but uses more credits.
Google Category — Browse all ~4000 official Google Business Profile categories. Useful when you're not sure what to search for — pick the exact category Google uses. Works like a single term search.
